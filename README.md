# NiceGallery

Приложение галерея для медиафалов (картинок и видео) под Android. Приложение дает пользователям возможность:

- удобно просматривать медиафайлы, организованные в древовидную иерархию папок (альбомов) с любым уровнем вложенности
- создавать, изменять, удалять папки (альбомы)
- переносить и удалять медиафайлы в папках (альбомах)
- приложение работает как с внутренним хранилищем смартфона, так и с SD-картами и USB накопителями
